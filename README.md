#Spring Data JAP startup Project

@Spring Boot @Spirng Data JPA

Spring Data JAP最简单的启动项目，涵盖了一个可供实战的最小Feature集合，包括
1. 基于方法名和@Query注解的查询方法， 
2. 分页和排序，
3. 用QueryByExample构建动态查询
4. 自定义数据访问的实现
5. TDD实战Reporsitory测试用例构建
